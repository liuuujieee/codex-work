# Xueying Doll

Snow-themed chibi doll pet based on Xueying Doll / 雪影娃娃 from Rock Kingdom World references.

## Files

- `pet.json` - Codex pet manifest.
- `spritesheet.webp` - Final 1536x1872 RGBA pet atlas.
- `qa/contact-sheet.png` - Visual QA sheet for all animation rows.
- `qa/validation.json` - Atlas validation output.

## Animation Rows

- idle
- running-right
- running-left
- waving
- jumping
- failed
- waiting
- running
- review
